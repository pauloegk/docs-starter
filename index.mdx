# API Manybot

Bem-vindo à documentação da API Manybot! Esta API permite que você integre e gerencie sua conta.

Estamos adicionando endpoints conforme a necessidade, se você precisar de algo específico, entre em contato conosco.

## Estrutura da API

A API está organizada em duas seções principais:

### 📨 Messages
Endpoints para envio de mensagens através dos canais conectados:
- **Send Text Message** - Enviar mensagens de texto

### 📱 Channels  
Endpoints para gerenciamento de canais de comunicação:
- **List Channels** - Listar todos os canais da empresa
- **Connect WhatsApp** - Conectar novo canal WhatsApp
- **Disconnect Channel** - Remover canal específico

## Autenticação

Todos os endpoints da API requerem autenticação via JWT no header:

Para usar a API, inclua o header `Authorization` com o valor `Bearer {seu-jwt-token}` em todas as requisições.

Para obter seu token JWT, acesse sua conta [App Manybot](https://app2.manybot.com.br) e clique em "Configurações" -> "Chaves de API".
